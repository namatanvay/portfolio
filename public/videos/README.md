# Videos Directory Structure

Place your videos in the appropriate folders:

## Music Videos
`music/` - Music video projects

## Travel
`travel/` - Travel videography

## Reels
`reels/` - Short-form content, Instagram reels, TikToks

## Commercial
`commercial/` - Commercial video work

## Video Guidelines
- **Format**: MP4 (H.264 codec) recommended for best browser compatibility
- **Resolution**: 1080p or 4K
- **Naming**: Use descriptive names (e.g., `wedding-highlight-reel.mp4`)
- **Size**: Keep file sizes reasonable (under 100MB per video if possible)
- **Thumbnails**: Consider adding poster images (same name as video but .jpg)
  - Example: `wedding-highlight-reel.mp4` and `wedding-highlight-reel.jpg`
